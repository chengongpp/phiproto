# PhiProto

PhiProto is yet another new message exchange format for AI bots. PhiProto aims at super security and privacy.

PhiProto file ends with .phiproto appendix.

## Usage

```bash
# Encode
phiproto -e plaintable.csv -o msg.phiproto
# Decode
phiproto -d msg.phiproto -o table.csv
```
